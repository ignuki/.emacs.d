## Dependencies
- `git`
- `ghc` and `cabal`


## Installation
1. Install dependencies.
2. Clone repo into `$HOME/.emacs.d`
3. Copy contents of `fonts` directory into `$HOME/fonts`
4. Run `fc-cache`
5. Run `emacs`.
6. `M-x all-the-icons-install-fonts`
7. Restart Emacs. You can now run E in regular mode or a a daemon.
